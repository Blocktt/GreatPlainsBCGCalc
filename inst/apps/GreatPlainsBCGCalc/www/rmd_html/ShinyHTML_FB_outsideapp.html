Prepare an input file outside of the app

The comma-separated file (CSV) needs to have all the required fields listed in the tables below. Column names must match the spelling, symbology and capitalization scheme exactly as shown.

IMPORTANT: Make sure you are using the proper Operational Taxonomic Unit (OTU) and taxa traits/attributes for your metric calculations. These can be found in the Taxa Translator and Attribute tables.

Input file requirements for bugs

Bug BCG Models Column Name  Data Type Domain (allowable entries) Description
All Index_Name text IA_Bugs_BCG, KS_Bugs_BCG, MO_Bugs_BCG, NE_Bugs_BCG Tells R which model to apply
All Index_Class text see table below for list of options Tells R which rules and metrics to apply, based on class
All SampleID text NA (flexible) Unique sample identifier (typically comprised of the site name, sample date, and replicate number)
All TaxaID text NA (flexible) Taxon name, using the BCG Operational Taxonomic Unit (OTU)
All N_Taxa numeric non-negative real number or for large/rare entries (Missouri), -99, 0 or blank Number of individuals
All NonTarget logical TRUE/FALSE FALSE = target (ok to include in calculations). TRUE = nontarget (not appropriate to include in the Great Plains BCG calculation)
All BCG_Attr text 2, 3, 4, 5, 6m Great Plains BCG taxa attribute assignments (2, 3, 4, 5, 6m). 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = highly tolerant; 6m = nonnative, moderately tolerant
All BCG_Attr2 text 4_better, 4_middle, 4_worse Great Plains BCG attribute IV sub-level: 4_better, 4_middle, 4_worse
All HABIT text BU, CB, CN, SP, SW Habit: burrowers (BU), climbers (CB), clingers (CN), sprawlers (SP), swimmers (SW). Multiple entries are allowed and are separated with a comma
All LIFE_CYCLE text uni, semi, multi [used in flag metrics] Life Cycle: number of broods or generations of an organism in a year (also referred to as ‘voltinism’): univoltine (UNI), semivoltine (SEMI), or multivoltine (MULTI). Multiple entries are allowed and are separated with a comma
All AirBreather logical TRUE/FALSE [used in flag metrics] TRUE = air breather, which include: Hemiptera (all families); Coleoptera - all adults other than Elmidae ande Dryopidae, and larvae of Dystiscidae, Hydrophilidae, Hydraenidae, Heteroceridae; Gyrinidae; Haliplidae; Scirtidae; Diptera (Culicidae larvae, Ptychopteridae larvae, Chaoboridae larvae (except Chaoborus sp.), Stratiomyidae, Dolichopodidae and Syrphidae
All Phylum text NA (flexible) Taxonomic hierarchy
All Class text NA (flexible) Taxonomic hierarchy
All SubClass text NA (flexible) Taxonomic hierarchy
All Order text NA (flexible) Taxonomic hierarchy
All Family text NA (flexible) Taxonomic hierarchy
All SubFamily text NA (flexible) Taxonomic hierarchy
All Tribe text NA (flexible) Taxonomic hierarchy
All Genus text NA (flexible) Taxonomic hierarchy
Iowa only BugGear text Qualitative, Surber, Hess, Artificial Substrate, Low-Flow Artificial Substrate and Anchored Artifical Substrate Iowa only. Used to differentiate Qualitative from Quantitative sample data prior to metric calculations.

Input file requirements for fish

Column Name  Data Type Domain (allowable entries) Description
Index_Name text GP_Fish_BCG Tells R which model to apply
Index_Class text see table below for list of options Tells R which rules and metrics to apply, based on ecoregion/size class
SampleID text NA (flexible) Unique sample identifier (typically comprised of the site name, sample date, and replicate number)
TaxaID text NA (flexible) Taxon name, using the BCG Operational Taxonomic Unit (OTU)
N_Taxa numeric non-negative real number Number of individuals
NonTarget logical TRUE/FALSE FALSE = target (ok to include in calculations). TRUE = nontarget (not appropriate to include in the Great Plains BCG calculation)
BCG_ATTR text 2, 3, 4, 5 BCG taxa attribute assignments for the Great Plains BCG: 2, 3, 4, 5. 2 = highly sensitive; 3 = sensitive ubiquitous; 4 = moderately tolerant; 5 = highly tolerant.
BCG_ATTR2 text 2_MIDDLE, 3_BETTER, 3_MIDDLE, 3_WORSE, 4_BETTER, 4_WORSE, 5_MIDDLE BCG attribute assignments that accommodate sub-levels: 2_MIDDLE, 3_BETTER, 3_MIDDLE, 3_WORSE, 4_BETTER, 4_WORSE, 5_MIDDLE.
BCG_NUM numeric 2.0, 3.0, 3.5, 4, 4.5, 5 Continuous numerical equivalent of BCG_ATTR2 field. 2_MIDDLE = 2.0; 3_BETTER = 3.0; 3_MIDDLE = 3.0; 3_WORSE = 3.5; 4_BETTER = 4; 4_WORSE = 4.5; 5_MIDDLE = 5.0. Used for the ‘x_BCG_Mean’ metric calculation.
NATIVE text NATIVE, NON_NATIVE Specifies whether a taxon is native or non-native to the Great Plains region as a whole
TROPHIC text HB, IV, OM, TC Trophic type: Herbivore (HB), Invertivore (IV), Omnivore (OM), top-carnivore (TC). A single taxon can only have one entry.
HABITAT text B, F, W Habitat type: benthic (B), fluvial (F), and water-column (W). A single taxon can have multiple entries.
REPRODUCTION text BEARER, BROADCASTER, COMPLEX NEST, LITHOPHIL, MIGRATORY, SIMPLE NEST Reproduction type. Only LITHOPHIL is used for metric calculations. A single taxon can have multiple entries.
FAMILY text NA (flexible) taxonomic hierarchy
GENUS text NA (flexible) taxonomic hierarchy
