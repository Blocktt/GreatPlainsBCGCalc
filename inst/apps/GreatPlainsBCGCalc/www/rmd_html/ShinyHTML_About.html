Background

This website calculates Biological Condition Gradient (BCG) scores for benthic macroinvertebrate (hereafter “bugs”) and fish samples collected from rivers in the Great Plains (Figure 1). The USEPA has published a practitioner’s guide to BCG development that would be useful introductory reading for someone new to the BCG framework (USEPA 2016). Details on the iterative development and validation of the Great Plains Bug and Fish BCG models found in the report by Jessup and Stamp (2024).

Want to try running the calculator but don’t have input files? Download test files.
